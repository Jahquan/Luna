TI-Nspire Converter

Luna web

Turn Lua, Python, XML, and BMP inputs into `.tns` documents in the browser, or drop a Python folder to recursively generate matching `.tns` files beside every script path in the virtual filesystem.

What stays the same

  • Native Luna conversion logic and XML handling stay untouched.
  • Single-output bundling still mirrors the CLI argument order.
  • Folder mode still only converts Python recursively.

This UI just makes the browser path clearer and easier to operate.

Drop files or a folder here

Drop loose `.lua`, `.py`, `.xml`, or `.bmp` files to build one `.tns`, or drop a Python folder tree to run recursive conversion.

Bundle mode Recursive folder mode Automatic downloads

Bundle files

Select one or more files and Luna will build a single `.tns` document from them.

If left blank, Luna uses the first selected file name with a `.tns` extension.

Convert folder

Pick a folder to recursively convert every `.py` file it contains into a same-name `.tns` output.

Folder mode ignores non-Python files and uses Luna's recursive CLI behavior.
